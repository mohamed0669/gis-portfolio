GIS Digitizing & Mapping

GIS Digitizing for Multiple Areas

Project Overview

This project focuses on GIS digitizing and mapping for multiple regions, converting raster and satellite data into accurate vector layers to support spatial analysis and planning.

Data Used

  • Satellite Imagery
  • Administrative Boundaries
  • Road Network (Line Data)
  • Land Parcels (Polygon Data)
  • Service Locations (Point Data)

Tools & Software

ArcGIS Pro Spatial Analyst QGIS

Methodology

  • Georeferencing raster and satellite images
  • Digitizing features (points, lines, polygons)
  • Attribute data creation and editing
  • Topology checking and error correction
  • Map layout design and visualization

Final Maps

Results & Conclusion

The project successfully produced accurate digital GIS layers for multiple areas, enhancing spatial data quality and supporting decision-making processes for urban and regional planning.
